The heck

Instant Page Messer let you plug custom effects to be possibly applied to pages as any build-in Instant Page Messer effects.

Instant Page Messer also provide useful pre-computed values to be used when programming such plugin, so they don't have to be computed again by each plugin.

How to

Building a custom plugable effect for Instant Page Messer

Minimal requirements

Any method (aka Javascript "function") that returns a booleanish value is a valid plugable effect for Instant Page Messer.

This booleanish value is used by Instant Page Messer to decide wether to call this method again as next animation frame or not. Truy values are for calling again; falsy are for not calling again. In all cases, Instant Page Messer may call a pluged effect anytime the user request messing with the displayed page.

This let programmers program one shot effects by always returning a falsy value. It also let them program animations that stop only under certain conditions.

Plug a custom effect to Instant Page Messer

Custom effect plugin is achieved by calling InstantPageMesser.plug() with an animation method (JavaScript "function") as only parameter.

Any JavaScript "function" is an animation method for Instant Page Messer as long as it returns true to run again and false to definitely stop.
